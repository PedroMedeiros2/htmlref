Descrição

A tag <!--...--> é usada para adicionar comentários no código HTML. Comentários são trechos de texto que não são exibidos no navegador e servem para incluir anotações, explicações ou lembretes para os desenvolvedores.

Exemplo

<!-- Este é um comentário. Ele não será exibido no navegador. -->

Notas

Os comentários em HTML são úteis para documentar o código e explicar partes específicas do documento. Eles podem ajudar outros desenvolvedores a entender melhor o propósito ou a estrutura do código.

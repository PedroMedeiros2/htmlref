Descrição

A tag <param> é usada em conjunto com a tag `<object>` para definir parâmetros ou configurações para objetos incorporados, como plugins de mídia ou elementos ActiveX em documentos HTML. Cada `<param>` especifica um nome e um valor para uma configuração específica do objeto.

Exemplo

<object data="video.swf" type="application/x-shockwave-flash">
    <param name="movie" value="video.swf">
    <param name="loop" value="true">
    <param name="autoplay" value="false">
    <param name="bgcolor" value="#FFFFFF">
</object>

Notas

A tag `<param>` é usada principalmente com objetos `<object>` que incorporam conteúdo de mídia, como vídeos ou animações em Flash.

Cada `<param>` deve ter um atributo `name` que especifica o nome do parâmetro e um atributo `value` que define o valor desse parâmetro.

O uso da tag `<param>` é menos comum em documentos HTML modernos devido ao declínio no uso de tecnologias como Flash e ActiveX. Em vez disso, recomenda-se o uso de tecnologias de mídia mais recentes, como HTML5 `<video>` e `<audio>`.

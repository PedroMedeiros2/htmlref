Tag <abbr>

Tag Anterior Referência HTML Completa Próxima Tag

Descrição

A tag <abbr> é usada para definir uma abreviação ou uma sigla no texto. Ela pode ser usada para fornecer uma explicação ou definição da abreviação quando o usuário passa o mouse sobre ela.

Atributos

A tag <abbr> pode conter os seguintes atributos:

  • title: Especifica o texto de explicação ou definição da abreviação, que será exibido como uma dica (tooltip) quando o usuário passar o mouse sobre ela.

Exemplo

<p>A <abbr title="World Health Organization">WHO</abbr> foi fundada
em 1948.</p>

Notas

O uso da tag <abbr> é útil para melhorar a acessibilidade e a compreensão do conteúdo por parte dos usuários, especialmente em casos de abreviações menos conhecidas.

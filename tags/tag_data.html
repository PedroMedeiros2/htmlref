Descrição

A tag <data> é usada para incorporar dados ou informações adicionais em um documento HTML, que são destinados principalmente a máquinas e processamento automático. Este elemento fornece um valor que pode ser lido por programas ou scripts, bem como um valor de substituição legível por humanos que é exibido no navegador.

Exemplo

<ul>
    <li><data value="21053">Chery Tomato</data></li>
    <li><data value="21054">Beef Tomato</data></li>
    <li><data value="21055">Snack Tomato</data></li>
</ul>

Notas

A tag <data> é especialmente útil quando você deseja incluir dados associados a elementos HTML sem afetar a apresentação visual no navegador.

Se os dados estiverem relacionados a datas ou horários, é mais apropriado usar a tag <time> em vez da tag <data>.

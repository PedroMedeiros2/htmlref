Referências HTML

O que é HTML e por que usá-lo?

O HTML (HyperText Markup Language) é a linguagem fundamental para a construção de páginas da web. Ele é responsável por definir a estrutura e o conteúdo de uma página, permitindo que os navegadores interpretem e exibam o conteúdo visualmente aos usuários. Ao criar uma página com HTML, você pode organizar o conteúdo em cabeçalhos, parágrafos, listas, links, imagens e muito mais.

HTML é a espinha dorsal da web, sendo a base de qualquer página que você vê online. Ao aprender e usar o HTML, você se torna capaz de criar suas próprias páginas web, personalizar seu layout, incorporar mídia e torná-las interativas através de formulários e outros elementos.

Como usar o HTML

Para começar a usar o HTML, você precisa de um editor de texto simples, como o Bloco de Notas (Windows) ou o TextEdit (Mac). Abra um novo arquivo e salve-o com a extensão ".html". A estrutura básica de um documento HTML é a seguinte:

<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Minha Página</title>
    </head>
    <body>
        <h1>Título da Página</h1>
        <p>Este é um parágrafo de exemplo.</p>
    </body>
</html>

Este exemplo mostra a estrutura básica de um documento HTML. Ele começa com a declaração do tipo de documento (DOCTYPE) e contém as tags <html>, <head> e <body>, que são os elementos principais do HTML. Você pode adicionar conteúdo dentro do <body>, como cabeçalhos, parágrafos e outros elementos, para criar sua página.

Elementos HTML

O HTML é composto por uma variedade de elementos, cada um representado por uma tag. Algumas das tags mais comuns incluem:

  • <h1> a <h6>: Títulos de diferentes níveis de importância.
  • <p>: Parágrafos de texto.
  • <a>: Links para outras páginas.
  • <img>: Imagens.
  • <ul> e <li>: Listas não ordenadas.
  • <ol> e <li>: Listas ordenadas.
  • <div>: Bloco genérico para agrupar elementos.
  • <span>: Container para texto sem efeito de linha.

Descrição

A tag <frameset> era usada em versões mais antigas do HTML, especificamente no HTML 4.0, para definir a estrutura de uma página da web com vários quadros ou frames. No entanto, essa tag foi obsoleta no HTML5 e não deve ser usada em documentos HTML modernos. Em vez disso, recomenda-se o uso de tecnologias como CSS, JavaScript e iframes para criar layouts de página complexos.

Exemplo (não recomendado)

<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">
<html>
<head>
    <title>Exemplo de Frameset</title>
</head>
<frameset rows="20%, 80%">
    <frame src="menu.html" name="menu">
    <frame src="conteudo.html" name="conteudo">
</frameset>
</html>

Notas

A tag <frameset> foi descontinuada no HTML5 em favor de abordagens mais flexíveis e acessíveis para criar layouts de página.

Para a criação de layouts de página com várias áreas, é recomendável usar elementos HTML semânticos, CSS e JavaScript para alcançar o mesmo resultado sem recorrer ao uso obsoleto do `<frameset>`. O uso de iframes também é uma opção para incluir conteúdo externo em uma página.

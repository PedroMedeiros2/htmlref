Referência da Tag <a>

Descrição

A tag <a> é usada para criar hiperlinks, permitindo que os usuários naveguem para outras páginas ou recursos relacionados.

Atributos

A tag <a> pode conter os seguintes atributos:

  • href: Especifica o URL de destino para onde o usuário será direcionado ao clicar no link.
  • target: Especifica como o link será aberto. Por exemplo, "_blank" abre o link em uma nova janela ou aba.
  • title: Define um texto de dica (tooltip) que aparece quando o usuário passa o mouse sobre o link.

Exemplo

<p>Visite nosso site: <a href="https://www.exemplo.com" target="_blank" title="Visite o site Exemplo">Exemplo</a></p>

Notas

Os links são uma parte fundamental da experiência na web, permitindo a navegação entre páginas e recursos relacionados. Certifique-se de usar a tag <a>com responsabilidade e fornecer URLs válidos e significativos para seus usuários.

Descrição

A tag <progress> é usada para representar a progressão de uma tarefa em andamento em um documento HTML. Ela é frequentemente usada para mostrar o progresso do carregamento de um arquivo, o avanço de um processo ou qualquer outra tarefa que tenha uma duração previsível.

Exemplo

<progress value="50" max="100">50% concluído</progress>

Notas

A tag <progress> tem dois atributos principais: value (que representa o progresso atual) e max (que define o valor máximo que representa a conclusão completa da tarefa).

Ao usar a tag <progress>, você pode informar aos usuários o status de uma tarefa em andamento, o que pode ser especialmente útil em aplicativos da web e carregamento de recursos.

Descrição

A tag <datalist> é usada em conjunto com a tag <input> para fornecer uma lista de opções sugeridas ao usuário enquanto ele digita em um campo de entrada de texto.

Exemplo

<input type="text" list="frutas">
<datalist id="frutas">
    <option value="Maçã">
    <option value="Banana">
    <option value="Laranja">
</datalist>

Notas

A tag <datalist> permite que você crie um conjunto de opções que será exibido ao usuário enquanto ele digita em um campo de entrada de texto. Cada <option> dentro do <datalist> representa uma sugestão para o usuário.

Essa funcionalidade é comumente usada em campos de pesquisa ou formulários para melhorar a experiência do usuário, fornecendo sugestões relevantes enquanto ele digita.

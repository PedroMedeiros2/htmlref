Descrição

A tag <label> é usada para associar um rótulo legível por humanos a um elemento de formulário HTML, como um campo de texto ou uma caixa de seleção. Isso melhora a usabilidade e a acessibilidade da página da web, permitindo que os usuários cliquem no rótulo para ativar ou focar no elemento de formulário correspondente.

Exemplo

<form>
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome"><br>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email"><br>
</form>

Notas

A tag <label> melhora a usabilidade de formulários da web, permitindo que os usuários cliquem no rótulo para focar ou ativar o campo de formulário correspondente.

O atributo "for" do elemento <label> deve corresponder ao "id" do elemento de formulário que está sendo rotulado para estabelecer a associação correta.

Usar <label> é uma prática recomendada para tornar formulários mais acessíveis a pessoas com deficiências visuais ou mobilidade limitada, além de melhorar a experiência geral do usuário.

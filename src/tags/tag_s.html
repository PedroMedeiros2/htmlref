Descrição

A tag <s> é usada para representar texto riscado ou obsoleto em um documento HTML. Ela é frequentemente usada para indicar que uma parte do texto não está mais em uso ou foi substituída por algo diferente.

Exemplo

<p>Este é um exemplo de texto <s>riscado</s>.</p>

Notas

A tag <s> é usada para fins de apresentação e não tem significado semântico específico. Ela é usada para indicar que um trecho de texto não está mais em uso, mas o navegador não faz nada para remover ou ocultar o texto riscado.

É importante observar que a tag `` não é usada para exclusão de texto, mas sim para mostrar que algo está riscado ou obsoleto.

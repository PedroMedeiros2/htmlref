Descrição

A tag <option> é usada em conjunto com a tag <select> para criar uma lista de opções em um menu suspenso ou caixa de seleção em um formulário HTML. Cada `

Exemplo

<select name="frutas">
    <option value="maca">Maçã</option>
    <option value="banana">Banana</option>
    <option value="laranja">Laranja</option>
</select>

Notas

A tag <option> é usada dentro de um <select> para criar as opções que o usuário pode escolher. O atributo value é usado para especificar o valor associado a cada opção, que será enviado para o servidor quando o formulário for submetido.

Os elementos <select> e <option> são frequentemente usados para criar menus suspensos e caixas de seleção em formulários da web.

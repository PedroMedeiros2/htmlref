Descrição

A tag <strike> é uma tag HTML obsoleta usada para representar texto riscado ou obsoleto em um documento HTML. Embora ainda seja suportada por navegadores, é recomendável usar a tag <s> em vez dela para indicar texto riscado.

Exemplo

<p>Este é um exemplo de texto <strike>riscado</strike>.</p>

**Aviso**: A tag <strike> é obsoleta em HTML5. Recomendamos o uso da tag <s> para representar texto riscado ou obsoleto.

Saiba mais sobre a tag <s>.

Notas

A tag <strike> não é mais recomendada em HTML5 e pode não ser suportada em versões futuras de navegadores. A tag <s> é uma alternativa mais apropriada para representar texto riscado.

É importante atualizar o seu código HTML para usar tags recomendadas e evitar o uso de tags obsoletas sempre que possível.

Tag <!DOCTYPE>

Tag Anterior Referência HTML Completa Próxima Tag

Descrição

A tag <!DOCTYPE> é usada para declarar a versão do documento HTML e o tipo de documento sendo usado. É uma instrução que informa ao navegador qual padrão de HTML está sendo utilizado no documento.

Sintaxe

<!DOCTYPE nome_tipo PUBLIC "URL_DTD" "nome_dtd">

Exemplo

<!DOCTYPE html>

Notas

O tipo de documento declarado pela tag <!DOCTYPE> afeta a forma como o navegador interpreta o código HTML, portanto, é importante usá-lo corretamente para garantir que o documento seja renderizado de acordo com as especificações desejadas.

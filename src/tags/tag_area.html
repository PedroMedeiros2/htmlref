Tag <area>

Tag Anterior Referência HTML Completa Próxima Tag

Descrição

A tag <area> é usada junto com a tag <map> para definir uma área sensível em uma imagem. Essas áreas sensíveis podem ser usadas para criar links ou mapeamentos de imagem.

Atributos

A tag <area> pode conter os seguintes atributos:

  • alt: Especifica um texto alternativo que é exibido quando a imagem não pode ser carregada.
  • coords: Especifica as coordenadas da área sensível no formato "x1, y1, x2, y2, ...". Dependendo do shape (forma), podem ser necessários diferentes números de coordenadas.
  • shape: Especifica o formato da área sensível. Pode ser "rect" (retângulo), "circle" (círculo), ou "poly" (polígono).
  • href: Especifica o URL de destino para onde o usuário será direcionado ao clicar na área sensível.
  • target: Especifica como o link será aberto. Por exemplo, "_blank" abre o link em uma nova janela ou aba.

Exemplo

<img src="imagem.jpg" alt="Imagem de exemplo" usemap="#mapa">
<map name="mapa">
    <area shape="rect" coords="0,0,100,100" href="pagina.html" target="_blank" alt="Área 1">
    <area shape="circle" coords="150,150,50" href="outra_pagina.html" target="_blank" alt="Área 2">
</map>

Notas

A tag <area> deve sempre estar dentro de uma tag <map>, e a imagem referenciada no atributo `src` deve ter a propriedade `usemap` apontando para o nome do mapa (no exemplo acima, `#mapa`). As coordenadas devem ser ajustadas de acordo com a posição e o formato desejado da área sensível na imagem.

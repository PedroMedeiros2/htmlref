Referência de Elementos HTML

Tag Definição
<!--...--> Comentário - Define um trecho de texto como comentário, que não é exibido na página e serve para anotações ou desativar temporariamente partes do código.
<!DOCTYPE> Declaração de tipo de documento - Especifica o tipo de documento e a versão do HTML sendo utilizada, ajudando o navegador a interpretar o código corretamente.
<a> Hiperlink - Cria um hiperlink para outra página ou recurso.
<abbr> Abreviação - Define uma abreviação ou acrônimo para um texto.
<address> Informações de Contato - Define informações de contato do autor ou dono do documento.
<area> Hiperlink de mapa de imagem - Define uma área clicável em um mapa de imagem usado como hiperlink.
<article> Artigo - Define um conteúdo independente e distribuível, como um post de blog ou uma notícia.
<aside> Conteúdo lateral - Define um conteúdo relacionado, mas não essencial ao conteúdo principal.
<audio> Arquivo de áudio - Incorpora áudio em um documento usando formatos suportados.
<b> Texto em negrito - Aplica estilo de texto em negrito.
<base> URL base - Define a URL base para URLs relativas em um documento.
<blockquote> Bloco para citações - Cria um bloco de citação que geralmente é recuado.
<body> Corpo do documento - Define o corpo do documento HTML.
<br> Quebra de linha - Insere uma quebra de linha (espaço) no texto.
<button> Botão - Cria um botão clicável para interação do usuário.
<canvas> Espaço reservado para gráficos dinâmicos - Utilizado para desenhar gráficos, animações e gráficos de forma dinâmica.
<caption> Legenda - Define uma legenda para uma tabela.
<cite> Usado para citações de trabalhos - Define uma referência a uma obra citada.
<code> Reservado para códigos - Define uma parte de texto que representa um código de computador.
<col> Coluna da tabela - Define uma coluna em uma tabela.
<colgroup> Grupo de coluna da tabela - Agrupa as colunas em uma tabela para estilização em conjunto.
<command> Comando - Define um comando que pode ser acionado pelo usuário.
<datalist> Opções predefinidas para outros controles - Define uma lista de opções pré-definidas para o elemento <input>.
<dd> Descrição ou valor - Define a descrição ou valor em uma lista de descrição.
<del> Texto excluído - Define o texto que foi removido ou deletado do documento.
<details> Controle para informações adicionais sob demanda - Define um elemento controlador que exibe informações adicionais quando clicado.
<dfn> Definindo instância - Define uma instância ou definição de um termo.
<div> Bloco genérico - Define um contêiner de bloco genérico para agrupar elementos.
<dl> Lista de descrições - Define uma lista de descrição (termo e descrição).
<dt> Termo ou nome - Define o termo ou nome em uma lista de descrição.
<em> Ênfase (itálico) - Aplica ênfase ou destaque em um texto (geralmente exibido em itálico).
<embed> Ponto de integração para plug-ins - Incorpora conteúdo externo ou plug-ins em um documento.
<fieldset> Conjunto de controles de formulário relacionados - Define um grupo de controles de formulário relacionados.
<figcaption> Legenda de uma figura - Define uma legenda para um elemento <figure>.
<figure> Figura com legenda opcional - Define uma figura que pode conter uma imagem ou ilustração com legenda opcional.
<footer> Rodapé (parte inferior da página) - Define o rodapé de uma página ou seção.
<form> Formulário - Define um formulário para entrada de dados do usuário.
<h1> to <h6> Título - Define títulos de diferentes níveis de importância (h1 sendo o mais importante e h6 o menos).
<head> Armazena metadados de documento - Define o cabeçalho do documento que contém metadados como título, scripts e links.
<header> Cabeçalho exibido na página - Define o cabeçalho da página ou seção.
<hgroup> Grupo de título - Define um grupo de títulos relacionados.
<hr> Linha horizontal - Cria uma linha horizontal para separar conteúdos.
<html> Elemento raiz - Define o elemento raiz do documento HTML.
<i> Itálico (atribui importância ao item) - Aplica estilo de texto em itálico.
<iframe> Permite que outras páginas sejam embutidas dentro de uma página - Incorpora um outro documento HTML dentro de um frame ou iframe.
<img> Imagem - Incorpora uma imagem no documento.
<input> Controle de entrada - Define um controle de entrada do usuário, como campos de texto, caixas de seleção ou botões.
<ins> Texto inserido - Define o texto que foi inserido no documento.
<label> Legenda para itens de formulários - Define uma legenda descritiva para um elemento <input> em um formulário.
<legend> Título ou legenda explicativa - Define uma legenda para um elemento <fieldset> em um formulário.
<li> Item da lista - Define um item de lista em uma lista ordenada (<ol>) ou não ordenada (<ul>).
<link> Permite relacionamentos entre documentos - Define um link entre o documento atual e um recurso externo, como um arquivo CSS ou um ícone.
<map> Definição de mapa de imagem - Define um mapa de imagem para usar com o elemento <area>.
<mark> Texto marcado (destacado) - Define o texto que foi marcado ou destacado no documento.
<menu> Permite a inserção de itens - Define uma lista de comandos ou opções.
<meta> Metadados - Define metadados que fornecem informações sobre o documento, como codificação e descrição.
<meter> Medidor escalar - Define uma medição dentro de um intervalo conhecido (por exemplo, medidor de bateria).
<nav> Grupo de links de navegação - Define um conjunto de links de navegação do documento.
<noscript> Carrega um conteúdo diferente caso o navegador não suporte scripts - Define o conteúdo a ser exibido caso o navegador do usuário não suporte scripts ou os scripts estejam desativados.
<object> Conteúdo externo genérico - Define um objeto incorporado, como um applet Java ou um objeto de mídia.
<ol> Lista ordenada - Define uma lista numerada ordenada.
<optgroup> Grupo de opções - Define um grupo de opções em um elemento <select>.
<option> Opção de uma caixa de seleção (select) - Define uma opção em um elemento <select>.
<output> Resultado do processamento de um formulário - Define o resultado de um cálculo ou processamento de formulário.
<p> Parágrafo - Define um parágrafo de texto.
<param> Parâmetros de inicialização para plug-ins - Define parâmetros para um plug-in em um elemento <object>.
<pre> Texto pré-formatado - Define um bloco de texto pré-formatado com espaçamento e quebras de linha preservados.
<progress> Indicador de progresso - Define uma barra de progresso para mostrar o status de conclusão de uma tarefa.
<q> Pequena citação de outra fonte - Define uma pequena citação de outra fonte.
<s> Armazena conteúdos que não são mais relevantes para o documento, sendo então eliminados - Define um texto que não é mais relevante ou preciso.
<script> Permite a adição de scripts - Define um script embutido ou um link para um arquivo de script externo.
<section> Bloco reservado para representar seções - Define uma seção lógica do documento, como capítulos, cabeçalhos ou outros temas.
<select> Caixa de seleção de um formulário - Define um menu suspenso para selecionar opções em um formulário.
<small> Letras pequenas - Define texto em tamanho de fonte menor.
<source> Permite várias fontes de áudio e vídeo - Define várias fontes de mídia para uso com o elemento <audio> ou <video>.
<span> Container para texto - Define um contêiner inline para agrupar elementos de texto sem efeito de linha.
<strong> Grande importância (negrito) - Define texto importante em negrito para maior destaque.
<style> Permite alterar o estilo - Define estilos CSS dentro do documento ou referencia um arquivo CSS externo.
<sub> Subscrito - Define texto que deve ser exibido ligeiramente abaixo da linha de base do texto.
<summary> Utilizado em sumários e legendas - Define um sumário ou legenda para um elemento <details>.
<sup> Sobrescrito - Define texto que deve ser exibido ligeiramente acima da linha de base do texto.
<table> Tabela - Define uma tabela para organizar dados em linhas e colunas.
<tbody> Grupo de linha de uma tabela - Define um grupo de linhas dentro de uma tabela.
<td> Célula de uma tabela - Define uma célula em uma tabela.
<textarea> Área para entrada de texto - Define uma área de texto multi-linha para entrada de dados do usuário.
<tfoot> Grupo de linha do rodapé de uma tabela - Define um grupo de linhas de rodapé em uma tabela.
<th> Célula de cabeçalho de uma tabela - Define uma célula de cabeçalho em uma tabela.
<thead> Grupo de título de uma tabela - Define um grupo de linhas de cabeçalho em uma tabela.
<time> Data e/ou hora - Define uma data e/ou hora específica.
<title> Título do documento - Define o título do documento que é exibido na barra de título do navegador ou na aba.
<tr> Linha de uma tabela - Define uma linha em uma tabela.
<track> Faixa de mídia suplementar - Define faixas de legenda, legendas ou descrições de áudio ou vídeo.
<u> Texto deslocado com estilo convencional com um sublinhado - Define texto sublinhado.
<ul> Lista não ordenada - Define uma lista de itens não ordenada.
<var> Variável - Define uma variável ou um termo em matemática ou programação.
<video> Vídeo - Incorpora um vídeo no documento.
<wbr> Insere uma quebra de linha (se necessário) - Define um local onde a quebra de linha pode ocorrer em uma palavra longa ou URL.
